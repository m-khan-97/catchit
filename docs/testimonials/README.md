# Testimonials

Real "this helped me" replies, captured as they happen. These compound over
time and can't be retro-collected — save one the moment it arrives, don't
wait to batch them up.

Also visa-evidence material (see `catchit-roadmap.md` §10): genuine,
dated, attributable impact statements are exactly what's hard to fabricate
after the fact.

## Adding one

Copy `TEMPLATE.md` to a new file named `YYYY-MM-DD-short-slug.md` and fill
it in. One file per testimonial.

**Only save a quote if the person has given permission to share it** —
ask explicitly if it isn't already clear from context (e.g. a public tweet
tagging the project counts; a private DM doesn't, unless they say it's fine
to quote).

Where they come from: the mailto link in the site footer ("Caught something
thanks to CatchIt? Tell us."), Discord, Reddit/community replies, or direct
messages from anyone using the site.
